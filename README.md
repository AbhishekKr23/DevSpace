# DevSpace 🌐💬

A Real-time Social Media Platform built for developers to share posts and interact.

## Tech Stack
- Node.js 🔥
- Express.js 🚀
- MongoDB 🍃
- Stripe 💳
- Socket.io 💬
- Passport.js 🔐
- Bootstrap 🎨

## Features
- User Authentication 🔑
- Real-time Chat 💬
- Post Creation 📝
- Like & Comment System ❤️💬
- Profile Management 👤
- Responsive Design 📱💻


## 🚀 Installation Steps
1. Clone the project:
git clone https://github.com/abhishek.kr23/DevSpace.git
cd DevSpace


2. Install Dependencies:
npm install

3. Run the Server:
node server.js

4. Visit:
👉 http://localhost:5000
